# llm-powered-financial-document-analyser
This Python project demonstrates the use of the Gemini API with Google Search Grounding and System Instructions to perform real-time, grounded financial analysis.

Technology Stack:
Python 3.x
requests: requests library to handle HTTP communication, specifically the necessary POST calls to the Gemini API endpoint.
ML Platform and Model, Google Gemini API (gemini-2.5-flash-preview-09-2025): Used Gemini 2.5 Flash as the projects core AI engine for its speed and reasoning capabilities.
